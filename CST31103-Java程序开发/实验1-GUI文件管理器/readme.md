# Java Exp1 GUI文件管理器  
## 一、实验目的
运用面向对象程序设计思想，基于Java文件管理和I/O框架，实现基于图形界面的GUI文件管理器。

## 二、实验项目内容  
1、实现文件夹创建、删除、进入。  
2、实现当前文件夹下的内容罗列。  
3、实现文件拷贝和文件夹拷贝（文件夹拷贝指深度拷贝，包括所有子目录和文件）。  
4、实现指定文件的加密和解密。  
5、实现指定文件和文件夹的压缩。  
6、实现压缩文件的解压。  
7、文件管理器具有图形界面。
